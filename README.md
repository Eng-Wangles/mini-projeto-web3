# 📦 Mini Projeto: Registrar Dados na Blockchain

## Objetivo
Demonstrar uma aplicação Web3 que registra dados (ex: "Produto enviado") em um contrato inteligente na blockchain Sepolia.

## ✅ Status do Projeto
- ✓ **Contrato Deployado**: [0x4dB20f580be4d5df4A560Eff8E581514674E4B7f](https://sepolia.etherscan.io/address/0x4dB20f580be4d5df4A560Eff8E581514674E4B7f)
- ✓ **Rede**: Sepolia Testnet
- ✓ **Interface**: HTML/JavaScript minimalista

## 🚀 Como Usar

### 1. Preparar o Ambiente
- Instale o [MetaMask](https://metamask.io/) no seu navegador
- Adicione a rede **Sepolia** no MetaMask
- Obtenha ETH de teste em: [Sepolia Faucet](https://www.alchemy.com/faucets/ethereum-sepolia)

### 2. Abrir a Interface
- Abra o arquivo `index.html` no navegador
- Você verá a interface com botão "Conectar Carteira"

### 3. Conectar Sua Carteira
1. Clique em **"Conectar Carteira"**
2. Aprove a conexão no MetaMask
3. Certifique-se de estar na rede **Sepolia**

### 4. Registrar um Dado
1. Digite um texto no campo "Dados do Produto" (ex: "Produto enviado - Pedido #123")
2. Clique em **"Registrar na Blockchain"**
3. Aprove a transação no MetaMask
4. Aguarde a confirmação

### 5. Verificar no Explorer
- O link para a transação aparecerá na interface
- Clique em **"Ver no Etherscan"** para confirmar
